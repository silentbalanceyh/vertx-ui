_sorter

内部模块

排序专用模块

排序函数中有两个维度的数据。

  1. 是否传入key：如果传入了key则证明是对对象中的字段为key的数据执行排序。
  2. 没有传入key：那么就是左右值的基础数据排序。

是否生成Function函数，如果生成Function函数，则通常用于 Ant Design中的 column 中的 sorter属性。

  1. 带Fn结尾的用于 column 的列排序。
  2. 不带Fn结尾的主要用于 Array 的 sort 方法排序。
