@ox(options)

Origin X注解

使用方法：

import Ex from 'ex';

@Ex.ox() -- 注释掉的调用方法，由于包含 @ 符号不可解析
class Component extends React.Component{

}

该注解用于高阶组件，使用Hoc方式对定义的React组件执行注解操作，采用ES7中的修饰语法

参数

options中目前包括两个核心值：

值 含义
Page 页面动态渲染。
Control 控件动态渲染。
