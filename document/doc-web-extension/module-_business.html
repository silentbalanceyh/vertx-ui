(static) dialog(reference) → {Object}

扩展函数

{
    add: (data) => Promise,
    save: (data) => Promise,
    saveRow: (data) => Promise,
    saveSelected: (data) => Promise
}

函数说明

函数名 说明
add 窗口添加执行
save 窗口保存执行
saveRow 保存当前行
saveSelected 保存当前窗口选择的

当前API的框架内部调用代码如：

import Ex from 'ex';

const $opSaveField = (reference) => params =>
     Ex.dialog(reference).saveRow(params);
